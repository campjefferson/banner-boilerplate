# Banner Boilerplate
Camp Jefferson banner creation boilerplate.

## Getting Started
`npm install`

## Usage

### CSS

#### Classes

`.ad-element`

`.frame`

#### Variables

`$bannerWidth`

`$bannerHeight`

#### Mixins

`@include size($width, $height)`

`@include position($x, $y)`

`@include koodo-legals($width, $position)`

### Build Commands

`npm start`

`npm run spritesheet`

`npm run production`

`npm run backups`

`npm run all`

## Built With
- SASS
- Handlebars
- ES6
